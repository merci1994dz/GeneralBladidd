# سجل تطوير تطبيق جنرال بلادي TV

تم تنفيذ التحديثات التالية على التطبيق بناءً على طلبك:

## 1. تحديث القنوات
- تم استيراد **4130 قناة** من الرابط المزود.
- تم تصنيف القنوات تلقائياً إلى الفئات التالية:
    - **رياضية**: تشمل قنوات beIN Sports و SSC وغيرها.
    - **جزائرية**: تشمل القنوات الوطنية والخاصة.
    - **إخبارية**: تشمل الجزيرة، العربية، وسكاي نيوز.
    - **أطفال**: تشمل كرتون نتورك و MBC 3.
    - **ترفيه**: تشمل قنوات MBC و OSN.
    - **دينية**: تشمل قنوات القرآن الكريم.
    - **وثائقية**: تشمل ناشيونال جيوغرافيك.

## 2. تحسينات واجهة المستخدم
- **إضافة ميزة البحث**: يمكنك الآن البحث عن أي قناة بالاسم مباشرة من الصفحة الرئيسية.
- **تحسين شريط التصنيفات**: تم دمج شريط البحث مع التصنيفات بشكل انسيابي.
- **تحسين الأداء**: تم تحسين استعلامات قاعدة البيانات للتعامل مع العدد الكبير من القنوات.

## 3. كيفية التشغيل
لتشغيل التطبيق في بيئة التطوير:
```bash
npm install
npm run dev
```

لإعادة استيراد القنوات في أي وقت:
```bash
npx tsx server/import_channels.ts
```

تم استخدام قاعدة بيانات SQLite (Drizzle ORM) لضمان السرعة والسهولة في التثبيت.
